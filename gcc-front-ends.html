Possible GCC front ends.

Common Lisp

  • CLiCC

    GPL. ANSI? Compiles to C. Last updated 1995.

  • OpenMCL

    LGPL. ANSI. Compiles to PowerPC machine code? Last updated October 2005 (1.0).

  • CLISP

    GPL. ANSI. Compiles to byte code. Last updated August 2005 (2.35).

  • CMUCL

    BSD License, public domain. ANSI. Compiles to machine code. Last updated October 2004 (19a-patch-002).

  • SBCL (derived from CMUCL)

    BSD License, public domain. ANSI. Compiles to machine code. Last updated October 2005 (0.9.6).

  • GCL

    GPL. CLtL1. Compiles to C. Last updated August 2005. Mail thread about GCL as a GCC front end.

  • ECoLisp

    License? ANSI? Compiles to C. Last updated September 1997.

  • ECL (derived from ECoLisp)

    License? ANSI? Compiles to C. Last updated November 2005 (0.9h).

  • MusCL

    Extremely liberal license. Unfinished. No compiler. Last updated July 2003.

To investigate further: integrates well with C (since the GCC back end is written in C)?

Scheme

  • Shoe

    GPL. R5RS. Compiles to x86 machine code. Last updated October 1999.

  • Guile

    GPL. Last updated December 2004 (1.6.7).

  • Scheme 48

    BSD-like licence. R5RS. Compiles to byte code. Last updated July 2005 (1.3).

  • Stalin

    GPL. R4RS. Compiles a single Scheme source file to C. Last updated June 2001.

  • MzScheme

    LGPL. R5RS. Compiles to byte code or C. Last updated December 2004 (209).

  • SCM

    GPL. R5RS. Compiles to C (with Hobbit). Last updated June 2005 (5e1).

  • Bigloo

    GPL, LGPL. R5RS. Compiles to C and Java. Last updated October 2004 (2.6f).

  • SIOD

    Unknown license. Unknown standard compliance. Last updated May 1997 (3.5).
